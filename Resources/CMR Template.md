# CMR Template

The **Comprehensive Marketing Review** — tier 2 of the [[Comprehensive Marketing Review|review ladder]]. A full, scored website + marketing deep-dive. Derived from the worked example [[Review — SAMA website 62-100]] (the proven format). Paid deliverable; ~50 min per site.

Scores against the [[Brand Resonance System]] and a stack of named frameworks (StoryBrand, CD Strategic Brief, Chet Holmes, Jay Abraham, CRO).

## Frontmatter
```
url: <site>
date: <YYYY-MM-DD>
assessor: <name>
depth: full
pages_analyzed: <n>
assessment_type: gaps_and_opportunities
```

## Structure

1. **Executive Summary** — Overall score `/100` + grade; Top 3 Strategic Opportunities (Priority / Impact / Effort); Quick Wins.
2. **The 8 scored dimensions** (each: score + table of findings + the opportunity):

| Dimension | Scale | Measures |
|---|---|---|
| Analytics & Tracking | /10 | Measurement maturity (GA4, GTM, Meta/LinkedIn pixels, Clarity…) |
| Social Media | /100 | Distribution — platform coverage & meta integration |
| StoryBrand Alignment | /20 | Narrative strength — customer-as-hero, villain, guide, plan, CTA |
| Strategic Positioning | /15 | Messaging clarity — ICP definition, differentiation, core promise |
| Messaging Hierarchy | /15 | Copy effectiveness — headlines, CTA density, benefit articulation |
| Conversion Architecture | /15 | Journey effectiveness — funnel logic, friction points |
| Trust & Authority | /10 | Credibility — social proof, testimonials, authority markers |
| UX & Technical | /10 | Experience quality — mobile, speed, SSL, navigation |

3. **Prioritized Action Items** — P0 (strategic/immediate) → P1 (quick wins) → P2 (long-term) → P3 (maintenance). Each: Action / Framework / Impact / Effort / Gap addressed.
4. **Framework-by-Framework Summary** — score + major gaps + opportunities per framework; name the top 3 fixes.
5. **Summary** — strengths, critical gaps, immediate actions, the strategic opportunity.

## Feeds into
The CMR's findings become the raw material for the [[SMS Template|Strategic Marketing Session]], which turns gaps into campaigns.
