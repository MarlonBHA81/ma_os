# Brand Messaging Session

Stage 2 of the [[Story Advantage delivery process]]. This is where **copy and strategy happen** — the review's findings get turned into the actual words the brand will use everywhere.

## Outputs

1. **[[Brand Messaging Playbook]]** — the core message assets (Brand Message, One-liner, Tagline).
2. **Channel copy**, written from the playbook:
   - Website copy
   - Email copy
   - SMS
   - WhatsApp
   - LinkedIn / Instagram outreach

Everything downstream ([[Wireframe page set]], [[Design stage]], [[Development tracks]]) implements the words produced here. Scores/positions against the [[Brand Resonance System]].
