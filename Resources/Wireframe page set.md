# Wireframe page set

Stage 3 of the [[Story Advantage delivery process]]. Wireframe the landing page(s) that **implement the full [[Comprehensive Marketing Review|marketing review]] plus the new [[Brand Messaging Session|messaging]]** — structure before visuals.

## The standard page set

- **Home**
- **About**
- **2–3 Services pages**
- **Booking page**
- **Booking success page**
- **Lead Magnet opt-in page**
- **Thank You page**

The wireframes translate review findings (e.g. weak trust signals, unclear journey) and the [[Brand Messaging Playbook]] into concrete page structure and flow. Feeds the [[Design stage]].
